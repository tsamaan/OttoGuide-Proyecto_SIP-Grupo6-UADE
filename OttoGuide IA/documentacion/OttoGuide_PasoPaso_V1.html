2.2
Crear Dockerfile propio para TTS
La imagen oficial rhasspy/wyoming-piper tiene un módulo Python faltante (pathvalidate) que causa que el contenedor crashee en loop. La solución es crear un Dockerfile propio que extiende la imagen oficial e instala el módulo faltante. Esto garantiza que la dependencia queda grabada en la imagen y no se pierde al recrear el contenedor.
👤 Teo
dockerfileservices/tts/Dockerfile
FROM rhasspy/wyoming-piper:latest # Fix: pathvalidate faltante en la imagen oficial # Sin esto el contenedor crashea con ModuleNotFoundError RUN /usr/src/.venv/bin/pip install pathvalidate
🐛 Bug encontrado en desarrollo · 01/05/2026
ModuleNotFoundError: No module named 'pathvalidate'
Síntoma: El contenedor ottoguide-tts arranca y entra en loop de reinicios continuos. dps muestra estado Restarting (1).
Causa: La imagen oficial rhasspy/wyoming-piper:latest no incluye el módulo pathvalidate en su virtualenv interno (/usr/src/.venv), pero el código lo importa al arrancar.
Lo que NO funciona: Instalar con docker exec ... pip install — la instalación se pierde cuando el contenedor se recrea. Tampoco funciona el pip del sistema por restricciones de Debian (externally-managed-environment).
Solución correcta: Crear un Dockerfile propio que use la imagen oficial como base y ejecute /usr/src/.venv/bin/pip install pathvalidate. Cambiar image: por build: en el compose. Así la dependencia queda grabada permanentemente en la imagen local.

3.4
Crear el script listen.py del wake word
Este script escucha el micrófono, detecta "Hola Otto" y notifica al orquestador. Es el primer paso del pipeline de voz.
👤 Lucas
pythonservices/wake-word/listen.py
import openwakeword import pyaudio import numpy as np import requests import os CORE_URL = os.getenv("CORE_URL", "http://core:8080") CHUNK = 1280 RATE = 16000 model = openwakeword.Model(wakeword_models=["hey_jarvis"]) # Modelo personalizado "hola otto" se agrega en Fase 4 p = pyaudio.PyAudio() stream = p.open(format=pyaudio.paInt16, channels=1, rate=RATE, input=True, frames_per_buffer=CHUNK) print("Escuchando wake word... (Hola Otto)") while True: audio = np.frombuffer(stream.read(CHUNK), dtype=np.int16) predictions = model.predict(audio) for name, score in predictions.items(): if score > 0.5: print(f"Wake word detectado! Score: {score:.2f}") requests.post(f"{CORE_URL}/wake", timeout=2) break
⚠ OpenWakeWord no tiene un modelo nativo de "Hola Otto". En esta fase usamos un modelo de prueba. El modelo personalizado se entrena y agrega en la Fase 4.

🐛 Bugs resueltos en desarrollo · 03-04/05/2026
LLM devuelve respuesta vacía — done_reason: length
Causa: num_predict: 150 era menor que los tokens del system prompt (~250). El modelo agotaba el presupuesto leyendo el contexto y no generaba respuesta. Solución: Eliminar num_predict del options — el Modelfile controla la longitud via instrucciones de respuesta corta.
Acentos corruptos en TTS — voz silenciosa o cortada
Causa: Pasar texto con acentos via echo "texto" | piper en bash dentro de Docker corrompe los caracteres UTF-8. Solución: Escribir el texto a /tmp/otto_texto.txt con encoding="utf-8" y usar cat archivo | piper.
Micrófono capta voz de Otto — feedback loop
Causa: arecord arrancaba antes que paplay terminara. Solución: Función esperar_fin_audio() que consulta pactl list sink-inputs hasta que el parlante queda libre.
Whisper transcribe en idiomas incorrectos (japonés, griego)
Causa: Parámetro language=es ignorado cuando se manda en el body. Solución: Pasarlo en la URL: ?language=es&task=transcribe. También se migró a modelo medium para mayor precisión.
UADE transcripto como "guadi", "huawei", "wai", etc.
Solución: Algoritmo de distancia de Levenshtein en similar_a_uade() — detecta automáticamente cualquier variación con hasta 2 errores fonéticos. Fallback manual disponible en services/core/correcciones_uade.json.
Voz con acento español — carlfm descartada
Solución: Migración a es_MX-gevy-high.onnx — voz masculina con acento latinoamericano, descargada del repositorio de la comunidad en Hugging Face (HirCoir).

import subprocess, requests, time, re, os # Puertos — cambiar según ambiente STT_PORT = 9001 # notebook desarrollo # STT_PORT = 9000 # robot Jetson LLM_PORT = 11434 TTS_PORT = 10200 STT_URL = f"http://localhost:{STT_PORT}/asr" LLM_URL = f"http://localhost:{LLM_PORT}/api/generate" # Dispositivo de audio MIC_DEVICE = "plughw:1,0" # notebook Linux Mint MIC_CHANNELS = "2" # MIC_DEVICE = "plughw:0,0" # robot Jetson # MIC_CHANNELS = "1" # robot Jetson WAKE_WORDS = [ "otto", "oto", "auto", "hola otto", "hola oto", "hola otto", "hola oto", "hola otto", "hola oto", ] def grabar(duracion, path): subprocess.run(["arecord", "-d", str(duracion), "-D", MIC_DEVICE, "-f", "S16_LE", "-c", MIC_CHANNELS, "-r", "16000", path], stdout=subprocess.DEVNULL, stderr=subprocess.DEVNULL) def transcribir(path): try: with open(path, "rb") as f: r = requests.post( f"{STT_URL}?language=es&task=transcribe", files={"audio_file": f}, timeout=30) return re.sub(r'[^\w\s]', '', r.text.lower().strip()) except Exception as e: print(f"[STT] Error: {e}") return "" def es_wake_word(texto): return any(w in texto for w in WAKE_WORDS) def preguntar_llm(pregunta): try: r = requests.post(LLM_URL, json={"model": "otto", "prompt": pregunta, "stream": False}, timeout=60) return r.json().get("response", "") except Exception as e: print(f"[LLM] Error: {e}") return "Perdona, no pude procesar eso." def hablar(texto): try: texto_esc = texto.replace('"', "'").replace('`', "'") subprocess.run(["docker", "exec", "ottoguide-tts", "sh", "-c", f'echo "{texto_esc}" | /usr/src/.venv/bin/piper ' f'--model /data/voices/es_ES-carlfm-x_low.onnx ' f'--output_file /tmp/respuesta.wav']) subprocess.run(["docker", "cp", "ottoguide-tts:/tmp/respuesta.wav", "/tmp/respuesta.wav"]) subprocess.run(["aplay", "/tmp/respuesta.wav"], stdout=subprocess.DEVNULL, stderr=subprocess.DEVNULL) except Exception as e: print(f"[TTS] Error: {e}") print("OttoGuide iniciado. Escuchando wake word 'Hola Otto'...") print(f"STT: {STT_URL} | LLM: {LLM_URL}") while True: grabar(duracion=3, path="/tmp/wake_check.wav") texto_wake = transcribir("/tmp/wake_check.wav") if not texto_wake: continue print(f"[WAKE] Escuche: '{texto_wake}'") if not es_wake_word(texto_wake): continue print("[WAKE] Hola Otto detectado!") hablar("Te escucho adelante") grabar(duracion=5, path="/tmp/pregunta.wav") pregunta = transcribir("/tmp/pregunta.wav") if not pregunta: hablar("No te escuche bien podes repetir") continue print(f"[STT] Pregunta: '{pregunta}'") respuesta = preguntar_llm(pregunta) print(f"[LLM] Respuesta: '{respuesta}'") hablar(respuesta) print("[LOOP] Volviendo a escuchar...") time.sleep(0.5)
